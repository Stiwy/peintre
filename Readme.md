# Peintre 

Peintre est un site internet permettant à un artiste peintre e publier ses œuvres.

## Environnement de développement 

### Pré-requis

* PHP 7.4
* Composer
* Symfony CLI
* Docker
* Docker-compose

Vous pouvez vérifier les pré-requis (sauf Doker et Docker-compose) avec la commande suivante (de la CLI Symfony) : 

>bash
symfony check:requirements

### Lancer l'"nvironement de développement 

>bash
docker-comose up -d
symfony serve -d
